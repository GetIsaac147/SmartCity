# City - Smart

Plataforma de una ciudad inteligente desarrollada por BeeStudents utilizando Tecnologias
para FrontEnd, UI/UX y acceso a algunas API's publicas.

## Caracteristicas

- Login y Registro de Usuarios
- Deshboard Responsivo
- Integracion con Firebase
- Consumo de APIs publicas (Clima, Noticias, Calidad del Aire)

## Tecnologia Utilizada (Stack de Desarrollo)

- HTML5
- CSS3
- Bootstrap 5
- Javascript
- Firebase
